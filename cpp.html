Descrição da Linguagem de Programação C++

C++ é uma linguagem de programação de propósito geral, desenvolvida por Bjarne Stroustrup nos anos 1980 como uma extensão do C. Ela suporta programação procedural, orientada a objetos e genérica, oferecendo recursos como classes, herança, polimorfismo e templates. C++ é amplamente utilizada em desenvolvimento de sistemas, software de alta performance, jogos e aplicações em tempo real devido à sua eficiência e controle sobre recursos de hardware. Além disso, a linguagem possui uma rica biblioteca padrão, a STL (Standard Template Library), que facilita o desenvolvimento de algoritmos e estruturas de dados.

// C++ code
int pot1 = 0, pot2 = 0, res = 0;
void setup()
{
pinMode(A0, INPUT);
Serial.begin(9600);
pinMode(A1, INPUT);
pinMode(12, OUTPUT);
pot1 = 0;
pot2 = 0;
res = 0;
digitalWrite(12, LOW);
}
void loop()
{
pot1 += analogRead(A0);
Serial.println("pot1-");
Serial.println(pot1);
pot2 += analogRead(A1);
Serial.println("pot2");
Serial.println(pot2);
if (pot1 > pot2) {
res += (pot1 + pot2);
Serial.println("res");
Serial.println(res);
digitalWrite(12, HIGH);
} else {
digitalWrite(12, LOW);
}
delay(1); // Wait for 1 millisecond(s)
}
