Java

Descrição da Linguagem de Programação Java

Java é uma linguagem de programação de alto nível, orientada a objetos, criada por James Gosling e sua equipe na Sun Microsystems em 1995. Conhecida por sua portabilidade, Java permite que programas sejam executados em qualquer plataforma que tenha a Java Virtual Machine (JVM), o que garante o princípio "escreva uma vez, execute em qualquer lugar.

A linguagem possui uma sintaxe semelhante à do C++, mas elimina muitos de seus recursos mais complexos, como ponteiros, priorizando segurança e simplicidade. Java é amplamente utilizada em desenvolvimento web, aplicativos móveis (especialmente Android), software corporativo e sistemas embarcados. Sua rica biblioteca padrão e um robusto ecossistema de frameworks, como Spring e Hibernate, tornam a linguagem popular entre desenvolvedores.


Entre suas principais características, Java é:

  • Multiplataforma: O código é compilado em um arquivo de código de bytes que pode ser executado em qualquer sistema com a JVM.
  • Orientado a objetos: Organiza o código em classes e objetos, facilitando a reutilização e manutenção do software. Mesmo após mais de 20 anos de uso, Java continua sendo uma das linguagens de programação mais populares, impulsionando desde aplicativos móveis até sistemas empresariais.

Exemplo de Código em Java (texto)

import java.util.Scanner;

public class programa02 {
public static void main(String[] args) {

Scanner sc = new Scanner(System.in);

int num = 0; int val = 0; int resposta = 0;
System.out.println("Digite o primeiro número: ");
num = sc.nextInt();
System.out.println("Digite o segundo número: ");
val = sc.nextInt();

resposta = num + val;

System.out.println("A soma é: " + resposta);
System.out.print("A soma é: " + resposta + "\n");
System.out.printf("A soma é: %d", resposta);

sc.close();
}
}
